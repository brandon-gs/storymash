This is a basic social network app using Javascript Technologies like Nodejs, Mongodb, React, and so on.

## Technologies

- Node
- Typescript
- Express
- React
- Next
- MongoDB

## Installation:

Make sure you have Node and NPM installed.

```bash
git clone https://github.com/brandon-gs/storybox
cd storybox
npm install
```

### Starting the app:

```bash
# Run in dev mode:
npm run dev

# Run in production:
npm run build
npm start
```
